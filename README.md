# stats

See your GitHub stats straight from the terminal with the power of Deno

# Installing

First, [install Deno](https://deno.land/).

Then:

```
$ git clone https://github.com/dosisod/stats
$ cd stats
```

# Running

```
$ ./path/to/deno run src/index.ts
```
